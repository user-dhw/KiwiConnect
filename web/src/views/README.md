# Views Structure

- `layout/`: Shell pages that host shared layout and nested routes.
- `home/`: Home and landing pages.
- `help/`: Q&A and help-related pages.
- `activity/`: Activity and event pages.
- `marketplace/`: Marketplace and second-hand pages.
- `news/`: Article and news pages.
- `search/`: Search result pages.
- `account/`: User account center pages.
- `support/`: Feedback, appeal, and report pages.

Use this structure for all new pages to keep routing and maintenance clean.
